## Getting Started

First, run the development server:

```bash
docker-compose build
docker-compose up

cd /frontend
npm install

npm run dev
# or
yarn dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
backend [http://localhost:8081](http://localhost:8081)
